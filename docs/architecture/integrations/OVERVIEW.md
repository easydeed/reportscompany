# External Integrations

> All third-party services used by TrendyReports

| Service | Purpose | Used By | Auth Method |
|---------|---------|---------|-------------|
| SimplyRETS | MLS listing data | Worker | Basic Auth |
| SiteX Pro | Property details + comparables | API | OAuth2 |
| Stripe | Subscriptions + billing | API | API Key |
| SendGrid | Email delivery | Worker | API Key |
| Resend | Transactional email (verification, reset) | API | API Key |
| Twilio | SMS notifications | Worker | Account SID + Auth Token |
| Cloudflare R2 | File storage (PDFs, images) | Worker | S3-compatible keys |
| PDFShift | Cloud PDF rendering | Worker | API Key |
| Google Maps | Places autocomplete + static maps | Frontend + Worker | API Key |
| OpenAI | AI market insights | Worker | API Key |
